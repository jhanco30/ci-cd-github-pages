## 📌 Descripción

Esta skill permite consultar de forma rápida y sencilla documentación técnica relacionada con Git, GitHub y CI/CD.  

Está diseñada para estudiantes y desarrolladores que necesitan recordar comandos, configuraciones o procesos sin tener que buscar en múltiples fuentes.

---

## 🎯 Problema que resuelve

Durante el desarrollo de software, es común olvidar comandos o configuraciones importantes (como git push, git pull, workflows, etc.).  

Buscar esta información en internet puede tomar tiempo y afectar la productividad.

Esta skill centraliza la información más importante para consulta inmediata.

---

## ⚙️ ¿Qué hace esta skill?

- Proporciona comandos esenciales de Git
- Explica configuraciones básicas de CI/CD
- Ofrece ejemplos listos para usar
- Reduce el tiempo de búsqueda de documentación
