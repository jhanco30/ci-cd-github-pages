## 🎯 Objetivo
Implementar un pipeline CI/CD funcional que automatice el despliegue de un sitio web usando GitHub Actions y GitHub Pages.

---

## ⚙️ ¿Qué es CI/CD?

- **CI (Integración Continua):**
  Permite integrar cambios de código automáticamente.

- **CD (Despliegue Continuo):**
  Publica automáticamente los cambios en producción.

---

## 🔧 Pipeline Implementado

### 🧱 Build
- Instalación de dependencias (`npm install`)
- Construcción del proyecto (`npm run build`)

### 🚀 Deploy
- Publicación automática en GitHub Pages
- Uso de GitHub Actions

---

## 🔄 Funcionamiento

1. Se realiza un cambio en el código
2. Se hace `git push`
3. Se ejecuta el pipeline automáticamente
4. Se despliega la aplicación
